TLDR: Rightclick "Install.inf" > choose "Install" > Select the cursor theme in the cursor preferences > OK/Apply > Enjoy!

2. Right click the "Install.inf" file, and choose "Install" in the menu. Discard any warning messages if they appear.
3. Go to the cursor options. (In Windows 10, search for "Themes" (in your language) and open themes. Then click the "Mouse Cursor" button.)
4. In the Mouse options, go to the "Pointers" tab.
5. Use the drop down menu to select the cursor theme and click OK. Agree with a message if it appears.
6. Enjoy!
